中国象棋 IDEA 运行问题解决方案
=======================================

【问题原因】
命令行编译的class文件是正确的（UTF-8编码）
IDEA编译的class文件是旧的（编码问题）
所以需要在IDEA中配置使用正确的编码

【快速解决方案】

步骤1：先运行 sync_to_idea.bat
       → 将正确的class文件复制到IDEA输出目录

步骤2：在IDEA中配置VM options
       1. Run → Edit Configurations...
       2. 找到主类配置
       3. VM options 输入：
          -Dfile.encoding=UTF-8 -Duser.language=zh -Duser.country=CN
       4. Apply + OK

步骤3：在IDEA中重新运行
       → 点击绿色的运行按钮

【完整解决方案】（如果上面不行）

1. File → Settings
2. Editor → File Encodings
3. 设置三项为 UTF-8：
   - Global Encoding: UTF-8
   - Project Encoding: UTF-8
   - Default encoding: UTF-8
4. Build → Rebuild Project

【验证】
运行后棋子应显示为：
红方：帅、仕、相、馬、車、砲、卒
黑方：将、士、象、马、车、炮、兵

【备用方案】
如果IDEA中仍然有问题：
→ 直接运行 final_fix.bat 即可
→ 或者使用命令行编译的版本

相关文件：
- IDEA_ENCODING_FIX.md (详细配置说明)
- idea_run_config.txt (VM options配置)
- sync_to_idea.bat (同步class文件脚本)
